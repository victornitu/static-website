# Static website compiler

**Pug/Sass/Coffee** => **Html/Css/ES5**

## Commands

### Install environment

`npm i` or `yarn`

### Compile once

`npm run build` or `yarn build`

### Auto Compile on change

`npm run build:watch` or `yarn build:watch`

### Clean compiled files

`npm run clean:build` or `yarn clean:build`
